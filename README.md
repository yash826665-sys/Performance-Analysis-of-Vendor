# Performance-Analysis-of-Vendor
I analyzed vendor's Performance using  Python, SQL and PowerBI.


An end-to-end data analytics project that analyzes vendor performance, procurement efficiency, inventory health, and profitability using **Python, SQL Server, and Power BI**. The project processes over **1.8 GB** of transactional data to generate actionable business insights.



## Tech Stack

- Python (Pandas, NumPy)
- SQL Server
- Power BI
- Jupyter Notebook

---

## Key Features

- Automated ETL pipeline using Python and SQL
- Interactive Power BI dashboard
- Vendor and brand performance analysis
- Inventory and profitability analysis
- Statistical analysis for business insights

---

| Metric | Value |

| Total Sales | **$450.02M** |

| Unsold Capital | **$8.35M** |


---

## Key Insights

- Top 10 vendors contribute **65.35%** of total procurement.
- Large purchase orders reduce unit costs by approximately **73%**.
- Low-volume vendors generate higher average profit margins than high-volume vendors.
- **$8.35M** of capital is tied up in unsold inventory, highlighting opportunities for inventory optimization. :contentReference[oaicite:2]{index=2}
